1. Selectors for state of elements.
    • :hover - selects an element when the user hovers over it with their mouse
    • :active - selects an element while it is being activated (e.g. clicked or tapped)
    • :focus - selects an element that has focus (e.g. when it is selected for input)
    • :enabled - selects an element that is enabled (i.e. not disabled)
    • :disabled - selects an element that is disabled
    • :valid - selects an input element whose value is valid (i.e. meets the specified validation criteria)
    • :invalid - selects an input element whose value is invalid (i.e. does not meet the specified validation criteria)
    • :read-only - selects an element that is read-only (i.e. cannot be edited by the user)
    • :read-write - selects an element that is editable by the user
    • :placeholder-shown - selects an input element that is displaying placeholder text
    • :default - selects a button that is the default button in a form
    • :valid - selects an option element that is the default option in a select element
  2. Form Selectors
    • :in-range - - selects an input element whose value is within a specified range
    • :out-of-range - selects an input element whose value is outside a specified range
    • :required -- selects an input element that is required to have a value before the form can be submitted
    • :optional - selects an input element that is not required to have a value before the form can be submitted
    • input[type=text] -- selects all input elements with a type attribute value of "text"
    • input[type=password] -- selects all input elements with a type attribute value of "password"
    • input[type=number] -- selects all input elements with a type attribute value of "number"
    • input[type=checkbox] -- selects all input elements with a type attribute value of "checkbox"
    • input[type=radio] -- selects all input elements with a type attribute value of "radio"
    • input:not[type=text] -- selects all input elements that are not a type attribute value of "text"
  3. Button Selectors
    • :hover -selector to change the style of a button when you move the mouse over it
    • :focus - selects a button that has focus
    • :enabled - selects a button that is enabled.
    • :disabled - selects a button that is disabled
    • :active - selects a button that is active(e.g clicked or tapped)
  4. Word Selectors
    • :contains() - selects an element that contains a specific word or phrase
    • :first-letter - selects the first letter of an element
    • :first-line - - selects the first line of an elementt
  5. Level 4 selectors
    • :dir() - selects an element based on its directionality (e.g. left-to-right or right-to-left)
    • :has() - selects an element if it contains a specified element
    • :matches() - - selects an element if it matches a specified selector
    • :not() - negates a selector, selecting elements that do not match the specified selector
    • :matches() - selects an element if it matches a specified selector
    • :nth-match()- selects an element based on its position in a list of elements that match a specified selector
    • :where() selects an element based on a condition (e.g. the value of an attribute or a computed style)
